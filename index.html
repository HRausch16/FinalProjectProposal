Description of Application:

I hope to have my application ask the user to input an animal and then draw a simple version of that animal. I also want to try to make the user select the color of the drawing.

Desired Features in Time Allowed:

  • User input animal
  • Draw animal if it is on list
  • Return message if animal isn't on list
  • Hosted online

Desired Feature if Given More Time:

  • User input color of drawing
  • Increase animal list
  • Allow users to input their own text files

Languages and Platform:

  • Python
  • Turtle Draw
  • HTML, CSS, and Java Script
  • Microsoft Azure

Technical Features from Class:

This application will demonstrate knowledge of various aspects of python including loops and user input. The application will also demonstrate knowledge of Turtle Draw and how it works.

Technical Features Outside of Class:

This application will require me to learn how to draw more complex and organized shapes in a text file. I will also need to figure out how to make Python read the proper file before reading it and drawing the image. I have also been doing research into utilizing Turtle Draw online, so I hope to use that research to allow others to use my application.
